SQL

テーブル： データのまとまり エクセルのシート
レコード： 行
カラム： 列
フィールド： エクセルでいうセル(カラムのことをいうこともある？)

リレーション： 関連
リレーショナルデータベース

・DDL： データ定義言語
        データベースの 作成 / 削除
        テーブルの 作成 / 削除 / 変更

・DML: データを操作するための言語
        データの 検索 / 挿入 / 更新 / 削除

・DCL： DMLやDCLをコントロールする言語
       トランザクションの 開始 / 終了 / キャンセル
        権限付与
        権限はく奪

・CRUD： DBRSに必要な4つの主な機能
       作成・読み込み・更新・削除
       Create Read Update Delete

       ・SELECT: テーブルのレコードを選択
                SELECT * FROM テーブル名;
       ・INSERT
        UPDATE DELETE

条件 
・データを検索するときに条件を指定する構文
  ・WHERE
  ・AND, OR, IN
  ・LIKE

その他
・その他、並び替えや件数制限、グループ化などの構文
  ・ORDER BY
  ・LIMIT, OFFSET
  ・GROUP BY

= : 完全一致
% : 部分一致に使う

※ すべて小文字でも動くが、SQLの正しい書き方は予約の文字は大文字にする
